Engelbrecht Family Budget

Your premium family financial dashboard — built with love for the Engelbrecht family.

⚙️ Setup Required:
To connect to your Supabase database, update the config at the top of the <script> section:

SUPABASE_URL — Your project URL
SUPABASE_KEY — Your anon public key

Then run the SQL schema in your Supabase SQL Editor and deploy to Vercel. See README.md for full instructions.

Demo mode uses browser localStorage — data persists until cleared
